# 🛒 Blinkit Sales Dashboard

A complete interactive Tableau dashboard project built using the Blinkit Grocery Dataset to analyze sales performance, customer ratings, outlet distribution, item categories, and product trends.

This project helps in understanding retail business insights through data visualization and analytics.

---

# 📌 Project Overview

The **Blinkit Sales Dashboard** is a business intelligence project created in **Tableau** for analyzing grocery sales data.

The dashboard provides detailed insights into:

- Total Sales Performance
- Outlet-wise Analysis
- Item Category Performance
- Fat Content Distribution
- Sales by Location Type
- Customer Ratings
- Product Visibility Analysis

The goal of this project is to transform raw grocery sales data into meaningful visual insights that support data-driven decision-making.

---

# 📊 Dashboard Features

## ✅ Key Performance Indicators (KPIs)

The dashboard includes important business metrics such as:

- 💰 Total Sales
- ⭐ Average Ratings
- 📦 Number of Items Sold
- 📈 Average Sales

## 📈 Visualizations Included

### 1. Sales by Outlet Type
Analyze how different outlet types contribute to overall sales.

### 2. Sales by Outlet Location Type
Compare sales performance based on outlet locations.

### 3. Sales by Item Category
Identify top-performing product categories.

### 4. Sales by Fat Content
Understand customer preference for low-fat and regular products.

### 5. Sales by Outlet Size
Evaluate how outlet size impacts revenue.

### 6. Item Visibility Analysis
Study the visibility impact on product sales.

### 7. Customer Ratings Analysis
Track product and outlet ratings.
